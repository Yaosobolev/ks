Тесты для самоконтроля

  1. 1) Какова цель присвоения IP-адреса интерфейсу VLAN1 на коммутаторе?

    Чтобы получить доступ к коммутатору для управления и конфигурирования;
    Чтобы коммутатор мог выполнять маршрутизацию между сетями;
    Для создания локальной сети с новым IP на коммутаторе;
    Чтобы коммутатор мог пересылать IP-пакеты;
    Для настройки безопасности коммутатора.

  2. Почему в корпоративной сети для передачи голосового трафика обычно требуется выделенная сеть VLAN?

    Она гарантирует сквозную передачу голосового трафика;
    Она гарантирует более высокую пропускную способность для голосового трафика;
    Она отделяет голосовой трафик от информационного, чтобы информационный трафик не влиял на голосовой;
    Она предоставляет поддержку резервного копирования голосового трафика, поскольку голосовой трафик более важен, чем информационный;
    Она обеспечивает более надежную передачу данных.

  3. Когда МАС-адреса удаляются из таблицы МАС?

    Через равные интервалы в 30 секунд;
    При получении широковещательного пакета;
    При изменении IP-адреса узла;
    После бездействия в течение определенного времени.
    Через равные интервалы в 60 секунд.

  4. Сколько времени необходимо порту коммутатора для перехода из состояния блокирования в состояние пересыпки при использовании STP?

    Одна секунда;
    15 секунд;
    20 секунд;
    50 секунд;
    60 секунд.

  5. Какое утверждение, связанное с функцией безопасности портов коммутатора, истинно?

    Коммутатор отключает неиспользуемые порты;
    Коммутатор фильтрует сетевой трафик, основываясь на МАС-адресах узлов;
    Коммутатор отключает порт, основываясь на номерах портов уровня 4;
    Коммутатор отключает трафик Telnet, поскольку имя пользователя и пароль передаются в виде обычного текста;
    Коммутатор фильтрует сетевой трафик, основываясь на IP-адресах узлов.

  10. Что произойдет, если записи из таблицы MAC-адресов не используются в течение определенного периода времени, называемого таймером устаревания?

    Коммутатор удаляет эти записи;
    Эти записи остаются до следующей перезагрузки коммутатора;
    Эти записи автоматически добавляются в ARP-таблицу;
    Эти записи автоматически добавляются в таблицу маршрутизации;
    Коммутатор назначает новое время таймера устаревания.
