Глоссарий

DAS – хранилище с прямым подключением. Система хранения данных, подключаемая непосредственно к узлам.

DCE – оборудование передачи данных. Физическое подключение к сети передачи данных в расширенной среде стандарта EIA. Оборудование передачи данных перенаправляет трафик и подает сигнал синхронизации, используемый для синхронизации передачи данных между устройствами оборудования передачи данных и оконечного оборудования. Примерами устройств оборудования передачи данных являются модем и интерфейсная плата. Оборудование передачи данных известно также под названием “оконечная аппаратура” при использовании в расширенной среде ITU-T.

DDoS – распределенная атака, вызывающая отказ в обслуживании. Атака через сеть с нескольких систем, занимающая полосу пропускания или ресурсы целевой системы.

DE – с правом отклонения. Обозначение пакета в сети Frame Relay. Пакет с битовым набором с правом отклонения будет отклонен первым при обнаружении маршрутизатором перегрузки сети. Бит права отклонения устанавливается в избыточном трафике, который является трафиком, полученным после установки набора гарантированной скорости передачи данных.

DHCP – протокол динамической конфигурации сетевого узла. Стандарт, используемый программным обеспечением, которое запрашивает и присваивает IP-адреса, адреса шлюза по умолчанию и адреса DNS-сервера сетевому узлу. DHCP-сервер выделяет IP-адрес для узла динамически, что позволяет повторно использовать ранее назначенный адрес после того, как он больше не нужен ему.

DLCI – идентификатор канала передачи данных. Адрес уровня 2, который необходим каждому виртуальному каналу для достижения получателя в NBMA-сети. Идентификатор канала передачи данных хранится в поле адреса каждого передаваемого кадра. Идентификатор канала передачи данных обычно имеет только локальное значение и может различаться в каждом оконечном устройстве виртуального канала.

DMZ – демилитаризованная зона. Специально предусмотренный участок между внутренней и внешней сетями.

DMZ (демилитаризованная зона) – специально предусмотренный участок между внутренней и внешней сетями (внешней сетью обычно является Интернет). Демилитаризованная зона используется устройствами, которые могут быть получателями трафика из Интернета: веб¬сервером, FTP-сервером, SMTP-сервером, DNS и т.п.

DNS – служба доменных имен. Система, преобразующая имена узлов из удобной формы URL в IP-адреса.

DoS – отказ в обслуживании. Атаки нескольких систем в одной сети, истощающая полосу пропускания или ресурсы целевой системы, например веб-сервера, с целью вывода ее из строя.

DRAM (динамическая память с произвольной выборкой) – это изменяющаяся память с произвольной выборкой в маршрутизаторах Cisco, состоящая из первичной DRAM, используемой для хранения таблиц маршрутизации, и общей DRAM, используемой для поддержки буферизации пакетов.

DROther – любой маршрутизатор в OSPF-сети, который не является назначенным или резервным.

DS1 – цифровой сигнал с уровнем 1. Спецификация кадрирования для передачи цифровых сигналов со скоростью 1,544 Мбит/сек в сетях Т1. используемых в США, или со скоростью 2,108 Мбит/сек в сетях Е1, используемых в европейских странах.

DS3 – цифровой сигнал с уровнем 3. Спецификация кадрирования для передачи цифровых сигналов со скоростью 44,736 Мбит/сек в сети ТЗ.

DSL (цифровая абонентская линия) – технология сетей общего пользования, позволяющая передавать данные с широкой полосой пропускания по медным кабелям на ограниченные расстояния. Способ непрерывного подключения пользователей к Интернету.

DSLAM (мультиплексор доступа к цифровой абонентской линии) – устройство, позволяющее двум или нескольким источникам данных использовать общую передающую среду. DSLAM разделяет сигналы телефонной связи и данные, передаваемые по DSL-линии, и направляет их в соответствующие сети.

DSO – цифровой сигнал с уровнем 0. Спецификация кадрирования для передачи цифровых сигналов через одиночный канал со скоростью 64 Кбит/сек в сети Т1.

DTE – оконечное оборудование данных. Физическое подключение к терминальному оборудованию пользователя в расширенной среде стандарта EIA. Оконечное оборудование данных служит в качестве источника данных, получателя данных или одновременно выполняет обе роли. Оно подключается к сети передачи данных через оборудование передачи данных, например, через модем, и обычно использует сигнал синхронизации, генерируемый оборудованием передачи данных. Примерами устройств оконечного оборудования данных являются компьютеры, трансляторы протоколов и мультиплексоры.

DTP – процесс передачи данных. Метод, устанавливающий соединение для передачи данных и управляющий им.

DUAL – алгоритм диффузионного обновления. Математический алгоритм, использующий EIGRP-протокол и обеспечивающий отсутствие петель путем непрерывного расчета маршрута. Алгоритм диффузионного обновления позволяет синхронизироваться маршрутизаторам, задействованным в изменении топологии сети, не затрагивая маршрутизаторы, которые не задействованы в изменении топологии сети.

DWDM – мультиплексирование с разделением по спектральной плотности. Процесс, который назначает входящие оптические сигналы определенным частотам или длинам волн луча света. Мультиплексирование с разделением по спектральной плотности может усиливать эти волны для увеличения мощности сигнала. Можно мультиплексировать более 80 различных длин волн или каналов данных в одном световоде оптоволоконного кабеля. Каждый канал способен передавать мультиплексированный сигнал со скоростью 2,5 Гбит/с.
