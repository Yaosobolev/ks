Глоссарий

BackboneFast – функция коммутаторов мостовой сети, которая обеспечивает быструю конвергенцию после изменения топологии связующего дерева. Функция BackboneFast используется на распределительном и центральном уровнях для восстановления соединения магистральной сети. Функция BackboneFast является запатентованной наработкой компании Cisco.

BCP – планирование непрерывности бизнеса. Действия, предпринимаемые для продолжения бизнес-процессов, в случае природных или антропогенных бедствий.

Be – избыточный пакет. Число бит, которые сетевое взаимодействие Frame Relay пытается передать после обработки предельного пакета. В общем случае, данные избыточного пакета доставляются с более низкой вероятностью, чем данные предельного пакета, так как данные избыточного пакета могут быть помечены сетью в качестве входящих данных. Избыточный пакет является номинальным показателем измерения для тарификации.

BECN – явное предуведомление о перегрузке. Сигнал в кадре, передаваемый в направлении, противоположном направлению передачи кадров, которые встретили перегруженный маршрут в сети Frame Relay. Оконечное оборудование, которое получает кадр с BECN-сигналом, может запросить протоколы более высокого уровня на выполнение необходимых действий по управлению потоками данных.

BGP – протокол BGP. Стандарт маршрутизации, используемый для соединения поставщика услуг с сетью Интернет. Протокол BGP известен также под названием протокол внешнего шлюза.

BID – идентификатор моста. Идентификационный код корневого моста, который является центральным узлом в STP-сети.

BOM (перечень материалов) – детализированный список, состоящий из аппаратных средств, программного обеспечения и других элементов, необходимых для построения сети. Перечень материалов используется для опредепения цены и заказа оборудования.

BPDU – блок данных протокола моста. Пакет приветствия протокола связующего дерева, который через заданный период времени пересылается для обмена данными между мостами в сети.

BSP – план обеспечения безопасности бизнеса. Физические, системы и организационные меры контроля, предпринимаемые для защиты сети и информационных ресурсов.

Bс – предельный пакет. Максимальный объем данных, выраженный в битах, который сетевое взаимодействие Frame Relay может принять и передать при гарантированной скорости передачи данных. Предельный пакет является номинальным показателем измерения для тарификации.
