Глоссарий

ABR (пограничный маршрутизатор области) – устройство маршрутизации, которое подсоединяет одну или несколько областей OSPF к магистральной сети. Пограничный маршрутизатор области поддерживает таблицы маршрутизации для магистральной сети и присоединенных областей OSPF.

ACL-список – список контроля доступа. Список, поддерживаемый сетевым устройством (например, маршрутизатором), для контроля доступа к маршрутизатору целого ряда служб. Например, ACL-список может быть использован для предотвращения отправки определенным интерфейсом маршрутизатора пакетов с заданным IP-адресом или номером протокола.

AD – объявленное расстояние. Расстояние, для которого широковещательная рассылка выполняется вышестоящим соседом.

AP – точка доступа. Устройство уровня доступа, подсоединяемое к проводной сети и используемое для передачи данных между проводными и беспроводными устройствами. Точка доступа соединяет беспроводные коммуникационные устройства для образования беспроводной сети с целью формирования роуминга.

ARP – протокол разрешения адресов. Стандарт, используемый для определения MAC-адреса узла при известном IP-адресе путем передачи ARP-запроса в сеть.

AS (автономная система) – объединение сетей с общим администрированием и общей стратегией маршрутизации. Автономные системы подразделяются на области. Автономной системе должен быть назначен уникальный 16-битный номер, устанавливаемый Комитетом по цифровым адресам в Интернете (IANA).

ASBR (пограничный маршрутизатор автономной системы) – пограничный маршрутизатор области, расположенный между автономной OSPF-системой и сетью, не поддерживающей OSPF-протокол. Пограничный маршрутизатор автономной системы использует OSPF-протокол маршрутизации и другие протоколы маршрутизации, например, протокол RIP. Пограничный маршрутизатор автономной системы должен постоянно располагаться не в оконечной OSPF-области.

ASCII – американский стандартный код для обмена информацией. Код с восьмибитовым представлением символов (7 битов + четность). Каждому алфавитному символу присвоен код от 0 до 127.

ASIC – интегральная схема прикладной ориентации. Схема, которая предоставляет точные инструкции для функционирования устройства во время коммутирования на третьем уровне модели OSI.

ASN – номер автономной системы. Идентификация AS для использования при маршрутизации пакетов сети Интернет.

ATM (асинхронный режим передачи) – международный стандарт для ретрансляции ячеек различных типов услуг, например, телефонии, видео или данных. При асинхронном режиме передачи службы передаются в ячейках фиксированной длины равной 53 байтам. Фиксированная длина ячеек уменьшает задержки передачи, так как обработка ячеек происходит на аппаратном уровне. Асинхронный режим передачи разработан для сред высокоскоростной передачи данных, например, ЕЗ, SO NET и ТЗ.
