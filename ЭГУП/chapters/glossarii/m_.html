MD5 Message Digest 5 – метод проверки подлинности, который требует, чтобы каждый маршрутизатор обладал уникальным ключом и идентификатором ключа. Маршрутизатор использует алгоритм, который обрабатывает этот ключ, OSPF-пакет и идентификатор ключа для генерации зашифрованного числа. Каждый OSPF-пакет содержит это зашифрованное число. Ключ никогда не передается.

MD5: Message Digests – Метод проверки подлинности, который требует, чтобы каждый маршрутизатор обладал уникальным ключом и идентификатором ключа. Маршрутизатор использует алгоритм, который обрабатывает этот ключ, OSPF-пакет и идентификатор ключа для генерации зашифрованного числа. Каждый OSPF- пакет содержит это зашифрованное число. Ключ никогда не передается.

MDF – главный распределительный отсек. Основная комната с коммуникациями в здании. Главный распределительный отсек является центральной точкой топологии звезда, в которой размещаются коммутационные панели, концентраторы и маршрутизаторы. Используется для присоединения открытых или частных каналов в здании к внутренней сети.

MetroEthemet – сетевая система, основанная на технологии Ethernet, охватывающая большой городе пригородами.

MIB – информационная база управления. База данных, содержащая сведения для управления сетью. Используется и обслуживается протоколом управления сетью, например SNMP или CMIP. Значение объекта таблицы MIB может быть изменено или найдено с помощью команд протокола SNMP или CMIP. Объекты таблицы MIB организованы в виде древовидной структуры, которая содержит общие или стандартные, и частные или патентованные, ветви.

MS Visio – Прикладное программное обеспечение для построения диаграмм, разработанное корпорацией Microsoft.

MTBF – среднее время наработки на отказ. Средняя продолжительность работы устройства без сбоев. MTBF используется для усредненной оценки отказоустойчивости жестких дисков и других компонентов компьютера.

MTTR – средняя продолжительность восстановления. Расчетное среднее время, которое будет затрачено на восстановление после обратимого отказа.

MTU – максимальный размер передаваемого блока данных. Максимальный размер пакета, в байтах, который может обработать конкретный интерфейс.
